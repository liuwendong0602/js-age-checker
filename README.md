JS Age Checker

一个基于 JavaScript 的前端交互项目，用于实现年龄判断与输入校验。

✨ 功能
输入年龄判断是否成年
输入校验（空值 / 非数字 / 非法值）
动态更新页面内容和样式
简单 UI 交互效果（按钮 hover、结果颜色变化）

🛠 技术栈
HTML
CSS（Flex 布局 + 基础 UI 设计）
JavaScript（DOM 操作、事件处理、条件判断）

🚀 项目亮点
使用原生 JavaScript 实现完整交互逻辑
实现输入校验与用户提示机制
通过 DOM 操作动态更新页面内容与样式
